목표

콘텐츠를 설계하고 구현하는 과정을 간단하고 빠르게 만듭니다.

주제

텍스트를 입력하여 알고리즘을 구현합니다. 텍스트의 종류와 특징을 알아봅니다.

텍스트의 종류

  1. 주석
  1. 변수
  1. 산술
  1. 몸통
  1. 함수
  1. 조건
  1. 반복
  1. 클래스

주석

프로그래밍 설계도, 즉 소스 코드에서 모든 텍스트는 컴퓨터가 인식해서 작동하지만, 주석은 프로그래밍에 있어 메모하는 목적으로 쓰입니다. 소스 코드를 더 쉽게 이해할 수 있게 만드는 것이 주 목적입니다.

한줄만 메모하고 싶을 때는 한줄 주석을 사용합니다. 한줄 주석은 슬래쉬 2개(//)로 작성하고 그 오른쪽은 모두 주석이 됩니다.

(소스코드..) //여기는 주석입니다 이 소스코드가 무엇인지 메모하기 위해 존재합니다

여러줄을 주석으로 작성하고 싶을 때는 여러줄 주석을 사용합니다. 여러줄 주석은 슬래쉬와 별표(/*)로 시작해서 별표와 슬래쉬(*/)로 끝나며, 그 사이가 모두 주석이 됩니다.

/*
	여기는 주석입니다 이 소스코드가 무엇인지 메모하기 위해 존재합니다
*/

변수

변수는 쉽게 말하면, 값을 담아놓는 바구니입니다. 임의로 설정한 A라는 문자, 즉 변수가 있다고 가정할 때, A의 값을 할당 또는 변경하고, 불러와서 가져올 수 있습니다.

먼저, 변수 A에 값을 할당할 때는 할당 연산자를 사용합니다. 할당 연산자는 수학에서의 같다(=) 기호 문자이고, 그 기호를 기준으로 오른쪽에 있는 것을 왼쪽 변수에 할당하는 의미입니다.

A = 777 //A변수의 값을 777로 할당합니다

산술

수에 대하여 간단한 계산을 하는 방법을 말합니다. 더하기, 빼기, 곱하기, 나누기, 나머지, 제곱, 총 6가지가 있습니다. 

A = 2 + 2  //더하기
B = 2 - 2  //빼기
C = 2 * 2  //곱하기
D = 2 / 2  //나누기
E = 2 % 2  //나머지
F = 2 ** 2  //제곱
G = (3 + 4) * 5  //응용

몸통

몸통은 쉽게 말해 텍스트 모아놓은 더미입니다. 텍스트끼리 서로 구분하거나 사용하기 위해서 묶어 놓은 것입니다 몸통 안쪽에서 바깥의 변수를 사용할 수는 있지만, 반대로 바깥쪽에서 안쪽의 변수를 사용할 수는 없습니다.

( ) { } [ ]  //모두 몸통을 의미하며 서로 다른 특징이 없습니다.
//단, 시작하는 기호와 끝나는 기호의 종류가 같아야 한 몸통을 이룹니다.

함수

  1. 함수 A를 사용하는 방법
  1. 함수 A의 알고리즘을 만드는 방법
A(숫자) {    //입력받은 값에 4를 더한 값을 출력하는 함수를 만듭니다
	return 숫자+4 
}

B() {        //10을 출력하는 함수를 만듭니다
	return 10
}

결과 = A(2)  //A 함수를 2라는 값으로 호출합니다
결과 = B     //A 함수를 호출할 때 몸통을 생략할 수 있습니다
b() {
	A = []  //빈 몸통 A를 할당
	A.add("Hello, World!")  //A에 Hello, World!라는 텍스트를 추가함
	A.add("bump")  //A에 bump라는 텍스트를 추가함
	A.remove("bump")  //A에 bump라는 텍스트를 삭제함
	return A
}

결과 = b

조건

맞느냐 맞지 않느냐라는 두가지 결과만이 존재하는 조건값은 참(true), 거짓(false)이 있습니다.

결과 = A < B  //왼쪽 값이 오른쪽 값보다 작을 때 참을 반환함
결과 = A <= B   //왼쪽 값이 오른쪽 값보다 작거나 같을 때 참을 반환함
결과 = A > B  //왼쪽 값이 오른쪽 값보다 클 때 참을 반환함
결과 = A >= B  //왼쪽 값이 오른쪽 값보다 크거나 같을 때 참을 반환함
결과 = A == B  //왼쪽 값과 오른쪽 값이 같을 때 참을 반환함
결과 = A != B  //왼쪽 값과 오른쪽 값이 다를 때 참을 반환함
결과 = !(A == B)  //부정연산자는 입력된 조건값을 반대로 반환함

두 텍스트 중 조건값에 따라 하나만 실행합니다.

if (A == B) {
	//조건이 참일 경우 실행되는 몸통
	결과 = '같습니다'
}
else {
	//조건이 거짓일 경우 실행되는 몸통
	결과 = '다릅니다' 
}
if (숫자 < 0) 결과 = '숫자가 음수입니다'
else if (숫자 == 0) 결과 = '숫자가 0입니다'
else if (숫자 > 0) 결과 = '숫자가 양수입니다'
else 결과 = '숫자가 아닙니다'
if (A == B && C == D && E == F) ...  //AND 연산자(&&)는 양쪽 값이 모두 참일 때 참
if (A == b || C == D || E == F) ...  //OR 연산자(||)는 양쪽 값중 하나라도 참일 때 참
if (A == B && C == D || E == F) ...  //A와 B가 같고 C와 D와 같거나, E와 F가 같을 때 참

반복

조건이 참일 동안만 텍스트를 반복합니다.

num = 0
while (num < 10) {  //값이 참일 경우 몸통을 실행하고 값이 거짓일 때까지 반복합니다
	...
	num = num + 1
}
//결과는 0에서 부터 9까지 입니다
//for(초기값, 조건값, 증감값)
for(i = 0; i < 10; i++) {
	...
}
//1. 초기값을 실행한다
//2. 조건값이 참이면 3으로 이동하고 거짓이면 4번으로 이동한다
//3. 몸통을 실행한 후 증감값을 실행하고 2로 이동한다
//4. 종료
